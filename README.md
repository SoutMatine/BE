# 📚 VocabBot — бот для изучения английских слов

Телеграм-бот на **Aiogram 3** с интервальным повторением слов и SQLite.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
export BOT_TOKEN="твой_токен_от_BotFather"
python bot.py
```

Или через `.env`:
```bash
BOT_TOKEN=твой_токен python bot.py
```

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и регистрация |
| `/add` | Добавить новое слово |
| `/review` | Повторить слова вручную |
| `/stats` | Статистика словаря |
| `/cancel` | Отменить текущее действие |
| `/help` | Справка |

## Алгоритм интервального повторения

Стадии (в часах): **1 → 6 → 24 → 72 → 168 → 336 → 720 → 1440**

- ✅ Ответил правильно → стадия повышается, следующий интервал длиннее
- ❌ Ответил неверно → стадия **не меняется**, интервал тот же
- 😔 «Я не помню» → стадия **не меняется**, интервал тот же

Бот автоматически присылает напоминания каждые 30 минут (если есть слова для повторения).

## Структура

```
vocab_bot/
├── bot.py          # точка входа
├── config.py       # настройки и интервалы
├── database.py     # работа с SQLite (aiosqlite)
├── handlers.py     # обработчики команд и FSM
├── scheduler.py    # планировщик напоминаний
└── requirements.txt
```
